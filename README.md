# Netflix Data Analysis using SQL

## Project Overview

This project analyzes the Netflix Movies and TV Shows dataset using SQL to uncover content trends, country-wise distribution, ratings analysis, and release patterns.

The objective is to perform exploratory data analysis (EDA) using SQL and generate business insights from Netflix's content library.

---

## Dataset Information

- Source: Kaggle
- Total Records: 8,800+
- Content Types:
  - Movies
  - TV Shows

## Key Insights

- Movies dominate Netflix's content library.
- The United States contributes the highest number of titles.
- Netflix experienced rapid content growth after 2015.
- TV-MA is one of the most common ratings.
- India is among the top content-producing countries.
